# README #

* gazebo simulator for ngws
* Version v1.0

* work with:
```
ROS indigo@ubuntu14.04 with gazebo 6
ROS kinetic@ubuntu16.04 with gazebo 7
```
* set up & Configuration
```
git submodule init
git submodule update
sudo apt-get install ros-indigo-ros-controllers
cd {workspace} 
catkin_make
```
